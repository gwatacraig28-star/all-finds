## How to deploy

1. Copy all files to a folder named `all-finds`.
2. Zip the folder.
3. Upload to a new GitHub repository (public).
4. Connect the repo to Vercel for deployment.
5. Add your Firebase API keys to `firebase-config.js`.
6. Add your Paynow and PayPal credentials to `payment-config.js`.
7. Set up your custom domain in Vercel.
